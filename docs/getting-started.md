# Getting Started with MCPSpec

## Installation

```bash
npm install -g mcpspec
```

## Quick Start

1. Initialize a project: `mcpspec init`
2. Edit `mcpspec.yaml` with your server and tests
3. Run tests: `mcpspec test`
4. Inspect a server: `mcpspec inspect "npx your-server"`

See the [README](../README.md) for full documentation.
